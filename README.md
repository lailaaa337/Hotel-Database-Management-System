# Hotel Database Management System

## Overview

This project is about designing a database system for managing hotel operations. It focuses on organizing and storing data related to guests, rooms, staff, and reservations in a structured way.

## What We did

* Designed an ERD to represent the system
* Created relational database schema
* Defined tables with primary and foreign keys
* Wrote SQL queries for table creation and data insertion
* Applied basic queries for data retrieval

## How it works

The database connects different entities like guests, rooms, staff, and bookings. This allows tracking reservations, managing room availability, and storing customer and payment information.

## Notes

This project helped me understand how real-world systems are modeled using databases and how relationships between entities are handled.
